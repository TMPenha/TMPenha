# 🔬 Research

MSc in Applied Computing for Data Science

## Dissertation

**Framework for Causal Observability and Remediation Recommendation for Data Pipelines using Data Lineage Graphs and Large Language Models**

The research proposes a framework capable of:

- modeling pipelines as **data lineage graphs**
- detecting **pipeline anomalies and failure propagation**
- identifying **causal dependencies across datasets and transformations**
- recommending **remediation strategies using LLM-based diagnostics**

### Research Goal

Build **explainable and self-diagnosing data infrastructures** capable of supporting reliable AI systems.
